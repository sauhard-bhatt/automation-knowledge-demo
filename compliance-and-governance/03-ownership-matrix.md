# Ownership Matrix

| Area | Responsible | Accountable | Consulted | Informed |
|---|---|---|---|---|
| Incident content | Platform maintainer | DevOps lead | App team SME | Support teams |
| Doc-gap content | Platform maintainer | DevOps lead | Onboarding owner | App teams |
| Migration runbooks | Platform enablement | DevOps lead | Security, app architect | App teams |
| Index maintenance | Platform maintainer | Repository owner | Reviewers | All users |
| Standards updates | Governance maintainer | DevOps lead | Senior maintainers | All maintainers |
| Release gate approval | Release approver | Repository owner | Reviewer | Contributors |

## Escalation

1. Conflicting technical guidance: escalate to DevOps lead.
2. Security concern: escalate to security reviewer.
3. Cross-team ownership dispute: escalate to repository owner.
